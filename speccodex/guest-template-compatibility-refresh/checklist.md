# Checklist

- [ ] `Windows 10/11` 文案已从主流程中移除，统一显示为 `Windows 10`。
- [ ] `Windows 10` 模板默认值已收敛为兼容安装优先，而不是 VirtIO 优先。
- [ ] `Windows 98` 模板默认值已收敛为旧系统兼容优先。
- [ ] 磁盘总线下拉框已出现 `IDE（兼容性）/ SCSI（冷门）/ SATA（稍快）/ VirtIO（快，半虚拟化总线，需安装驱动）`。
- [ ] `sata` 不只是前端文案，运行时也能真实映射。
- [ ] 显卡选项已包含 `std`、`qxl`。
- [ ] 网卡选项已包含 `ne2k_pci`。
- [ ] `Windows 10` 默认网卡已改为 `rtl8139`。
- [ ] `Windows 98` 默认网卡已改为 `pcnet`。
- [ ] 显示前端在主流程中已锁定为 `Sanaka`。
- [ ] `Sanaka 连接协议` 已改成固定 `VNC`，不再作为可选项。
- [ ] 主流程中不再把 `SPICE` 暴露成已完成的正式显示前端。
- [ ] 音频文案已改成宿主机音频后端心智，不再暗示 `VNC` 自带成熟音频通道。
- [ ] 高级选项已出现 `UEFI` 左右开关。
- [ ] 未配置可用固件时，`UEFI` 启动不会伪成功。
- [ ] 旧 `template.key = "win11"` 的机器仍能兼容读取。
