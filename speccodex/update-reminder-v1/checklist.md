# Checklist

- [ ] 远程更新清单使用 TOML。
- [ ] 更新清单至少包含 `version`、`channel`、`mandatory`、`pub_date`、`url`、`notes`。
- [ ] 本地 `release` 版本不会收到 `beta` 提醒。
- [ ] 本地 `beta` 版本会收到 `beta` 和 `release` 提醒。
- [ ] 启动后会静默检查更新。
- [ ] 应用运行期间每 8 小时会主动检查更新。
- [ ] 设置页有手动“检查更新”入口。
- [ ] 有更新时会显示更新提醒弹层。
- [ ] 弹层中能看到版本号、发布时间和更新内容。
- [ ] 更新内容过长时内容区可滚动。
- [ ] 点击“前往下载”会打开默认浏览器。
- [ ] 点击“稍后”只关闭当前提醒。
- [ ] 点击“跳过此版本”后，同版本不再提醒。
- [ ] 更高版本出现后，提醒会恢复。
- [ ] 弹层进入时不会闪，不会鼠标动了才显示，不会直接切到最终态。
