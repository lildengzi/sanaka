# Checklist

- [ ] 机器配置里存在共享文件夹默认结构，不会破坏旧 `.svm` / `.saka`。
- [ ] 共享文件夹只在 `user` 网络模式下作为正式主路径启用。
- [ ] 启动虚拟机时，QEMU 共享参数不会和现有网络参数冲突。
- [ ] 运行态能告诉前端：是否启用、是否生效、是否需要重启后生效。
- [ ] 宿主机缺少共享能力时，用户能看到明确提示，而不是静默失败。
- [ ] Windows 现代客户机能访问共享目录。
- [ ] 宿主机和客户机都能在共享目录中创建、删除、修改普通文件。
- [ ] 运行中切换目录后，不会假装立即生效。
- [ ] 老系统模板不会被错误标记为“完全支持”，但也不会被硬禁止。
- [ ] 给 Kimi 的前端对接需求已单独写清楚。
