# Checklist

- [ ] 每个机器包内部都有正式 `Disks/` 目录。
- [ ] 新建磁盘不会再落到随机工作目录或只存在于内存草稿里。
- [ ] `machine.svm` 中不会再出现没有语义的裸文件名。
- [ ] 导入磁盘后，默认被标记为外部磁盘引用。
- [ ] 新建磁盘默认位于机器包内部。
- [ ] Runtime 能正确区分 `managed` 和 `external` 磁盘。
- [ ] Runtime 启动前会先检查磁盘入口是否存在。
- [ ] 机器包移动后，包内创建型磁盘仍能启动。
- [ ] 外部磁盘在来源仍存在时可以正常启动。
- [ ] 执行“整理到虚拟机”后，整台机器可单独复制到其他机器。
- [ ] 不再出现 `file=czadb.qcow2` 这类裸文件名直接交给 QEMU 的情况。
