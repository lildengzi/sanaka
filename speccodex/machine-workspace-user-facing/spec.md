# Machine Workspace User-Facing Cleanup

## Goal

把首页和详情页从内部信息展示收回到用户能理解的虚拟机对象界面。

## Decisions

- 首页不再展示“其他虚拟机”解释性卡片区
- 其他虚拟机主要通过左侧对象列表切换
- 侧栏机器列表按机器 `id` 去重
- 详情页不显示内部文件路径
- 详情页不显示“运行状态”和“最近活动”侧栏
- 机器运行状态使用用户能理解的状态文案
- 首页和详情页未接 Runtime 前默认显示 `未启动`，不把操作词 `关机` 当静态状态展示
- 控制台内允许使用 `关机`，因为控制台是运行控制场景
- 关机主操作显示 `启动虚拟机`
- 开机后主操作显示 `进入虚拟机`

## User-Facing Rule

界面不能暴露内部路径、内部配置文件名、开发阶段解释或文档用语。用户看到的是虚拟机对象和操作，不是实现细节。
